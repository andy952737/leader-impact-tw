# LeaderImpact 台灣首頁 UI 計劃

> 建立日期：2026-04-21  
> 負責團隊：LeaderImpact 台灣

## 技術規格

- **語言**：純 HTML5 + CSS3 + 原生 JavaScript（不使用任何框架）
- **語系**：繁體中文（台灣）
- **LOGO**：直接引用 `https://www.leaderimpact.com` 的圖片 CDN 連結
- **響應式設計**：支援桌面、平板、手機三種斷點

---

## 頁面結構（由上至下）

### 1. 導覽列（Navbar）

```
[ LOGO ]   小組  關於  資源  NEXT  聯絡   [ 會員中心 ]  [ 領袖中心 ]  [ 捐款 ]
```

- 固定置頂（sticky）
- 左側：LeaderImpact LOGO（引用自官網 CDN）
- 中間：導覽連結（繁體中文）
- 右側：三個功能按鈕（不同顏色區分）
- 手機版：漢堡選單

### 2. Hero 英雄區塊

- 全寬背景圖（深色半透明遮罩）
- 大標語：**「倍增您的影響力」**（Multiply Your Impact）
- 副標：「加入我們遍佈全球、渴望留下真實影響力的領袖社群。」
- 三個 CTA 按鈕（橫排）：
  - `尋找小組`（主要按鈕，橘色）
  - `開始小組`（次要按鈕，描邊）
  - `我加入`（第三按鈕，白底）

### 3. 最新資源（The Latest）

- 小標題：`資源`
- 大標：**「最新消息」**
- 三張資源卡片（水平排列，手機版縱列）：
  - **影響力領袖影片系列**（BALOI Video Series）— 連結至官方影片頁
  - **《領袖整合人生》**（The Integrated Life of Leaders）— 書籍介紹
  - **《成為影響力領袖》**（Becoming a Leader of Impact）— 書籍介紹
- 每張卡片：封面圖 + 標題 + 簡短描述 + 「了解更多」連結
- 區塊底部：「更多小組資源 →」按鈕

### 4. 影響力故事（Impact Stories）

- 大標：**「影響力故事」**
- 三張人物故事卡片：
  - Eni Xheko（阿爾巴尼亞）
  - Derek Yeung（香港）
  - Mark Hawn（美國）
- 每張卡片：人物縮圖 + 姓名 + 「他/她的故事 →」按鈕
- 精選引言輪播（Carousel）：
  - 顯示一段引言 + 人名 + 職稱
  - 左右切換箭頭
- 引言（繁中翻譯版）：
  > 「這就是領袖影響力社群的本質——讓我們真正相信彼此、信任彼此。」  
  > — Eni，市場領袖，阿爾巴尼亞

### 5. 行動呼籲橫幅（CTA Banner）

- 深色背景橫幅
- 大標：**「您是百萬領袖之一，立志留下永恆影響力嗎？」**
- 橫排按鈕群：
  - `訂閱電子報`
  - `尋找小組`
  - `開始小組`
  - `資源`
  - `關於我們`

### 6. 頁腳（Footer）

- 四欄網格：快速連結 / 帳號入口 / 法律 / 版權
- 版權文字：「©1994-2024. LeaderImpact™ 版權所有。」
- ECFA 認證標章（佔位符）
- `隱私權政策` | `使用條款`

## 備注

- LOGO 直接以 `<img>` 標籤引用官方 CDN，若連結失效自動切換至本機備援 SVG
- 所有外連（官方資源頁、Amazon 書籍頁）保留原始 URL
- Hero 背景圖使用 Unsplash 免費商用圖（若官網圖片無法跨域使用）
- 引言輪播支援：自動播放（5秒）、手動切換、觸控滑動、鍵盤操作
